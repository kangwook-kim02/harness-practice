# PRD: 대학생 단기 숙박 공유 플랫폼

## 목표
대학생이 방학·단기 공석 방을 안전하게 내놓고 구할 수 있는 대학생 전용 숙박 공유 플랫폼

## 사용자
- 대학생 (학생증 또는 재학증명서로 인증된 사람만 이용 가능)
- 하나의 계정으로 매물 등록(집주인 역할)과 검색(구하는 사람 역할)을 모두 수행 가능

## 핵심 기능
1. 대학생 인증 — 학생증 또는 재학증명서 업로드 후 관리자 수동 승인
2. 매물 등록 — 위치, 가격(1박 기준), 숙박 가능 기간, 집주인 동의서(파일), 사진 업로드
3. 매물 검색 및 정렬 — 키워드 검색, 가격 낮은순/높은순 정렬, 기간 필터
4. 플랫폼 내 메시지 — 매물 기준 1:1 대화 (연락처 미노출)

## MVP 제외 사항
- 결제 시스템 (예약금, 정산)
- 리뷰 및 평점
- 지도 기반 검색 (지도 API 연동)
- 예약 확정 플로우 (수락/거절 상태 관리)
- 이메일/푸시 알림

## 디자인
- 라이트 모드 고정, 신뢰감 있는 미니멀 디자인
- 색상: 화이트 배경, 그레이 계열 텍스트, 포인트 컬러 1가지 (블루 계열)
